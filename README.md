# Weather App - تطبيق الطقس

![Weather App Screenshot](./public//assets/weatherprojects.png)

## وصف المشروع

تطبيق ويب بسيط لعرض حالة الطقس في مدينة محددة (مثلاً صنعاء)، مع دعم:

- اللغتين العربية والإنجليزية.
- عرض درجة الحرارة الحالية، الحد الأدنى والأقصى.
- وصف حالة الطقس بالأيقونات والنصوص.
- تحديث الوقت والتاريخ حسب اللغة.
- واجهة متجاوبة باستخدام Material-UI.

---

## 🛠️ التقنيات المستخدمة

- **React.js** - لبناء واجهة المستخدم.
- **Material-UI (MUI)** - لتصميم مكونات واجهة جميلة ومتجاوبة.
- **Axios** - لجلب بيانات الطقس من API.
- **Moment.js** - لتنسيق الوقت والتاريخ حسب اللغة.
- **i18next + react-i18next** - لدعم الترجمة بين العربية والإنجليزية.

---

## 🌍 مميزات المشروع

1. **دعم اللغة**: يمكن التبديل بين العربية والإنجليزية بضغطة زر.
2. **حالة الطقس**: تعرض أيقونة الطقس، الوصف، والدرجات (الصغرى والكبرى).
3. **تاريخ ووقت مباشر**: يعرض الوقت الحالي بتنسيق مناسب حسب اللغة.
4. **واجهة متجاوبة**: تتوافق مع جميع أحجام الشاشات.
